News Highlight

This is a Python Flask Application that allows users list and preview news articles from various sources. It consumes Newsapi.org API Key to collate news from various sources and users can visit the specific news links to read more.

Web Scraper

Mtaa Watch

This is a Python application based on Django framework that allows you to be in the loop about everything happening in your neighborhood. From contact information of different handyman to meeting announcements or even alerts.

Multibank Calculator

This is JS, jQuery, Bootstrap, HTML5, CSS3 application allows users to view loan products and compare across various banks for the cheapest and most flexible terms.

GitHub Search

This is an Angular 7 application that consumes GitHub API and allows users to enter a GitHub username into a form, submit it and see names and descriptions of that person's public repositories and other GitHub related information.

Communication Microservice

This is a Python Django application that consumes both Twilio and Sendgrid API to create a communication microservice that uses Dramatiq for message scheduling and Redis for message brokering

Password Manager

This is a Python application that will help you to manage your passwords and even generate new passwords. On Average, a person has at least 7 different accounts he or she has signed into, be it email, social media, entertainment or job portal accounts.

Pig Dice

This is a JS, jQuery, Bootstrap, HTML5, CSS3 game which is playable with just a single die. The first player to score 100 or more points wins the game. Each turn, a player repeatedly rolls a die until either a 1 is rolled or the player decides to "hold".
